## 1.2 case语句

与常规的case语句不同，进入一个case后，不需要break语句防止流程进入后面的case语句

```go
switch getStatus() {
case "statusA":
  // do A
  // 不需要break语句，switch分支自动退出
case "statusB":
  // do B
default:
  // do default
}
```
