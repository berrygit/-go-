## 变量

**指针**

指针是变量的地址，变量一定有对应的指针，指针通过`&`获取，如

```go
var x int         // x是变量
p := &x           // p是变量x的指针（地址），p的类型为*int
fmt.Println(*p)   // 可以通过*p获取指针指向地址保存的内容，也就是x的值
```

在go语言中，指针存在的主要作用是用来解决函数调用时值传递的问题，提供一种手段跨函数修改变量中的内容。
